# Игра «Кто хочет стать миллионером?» на Python (Tkinter)

Проект — настольная версия знаменитого телешоу «Кто хочет стать миллионером?» с современным графическим интерфейсом на Tkinter.  
Игроку предлагаются вопросы с четырьмя вариантами ответов и растущей шкалой призов — от символических сумм до виртуального миллиона.

## Основные возможности

- Современный интерфейс на Tkinter: аккуратные окна, кнопки и подсветка вариантов.
- Отдельные модули для логики и интерфейса: код легко поддерживать и развивать.
- База вопросов в формате `JSON`: можно быстро добавлять и изменять вопросы.
- Логика уровней сложности и призовой лестницы.
- Скрипты для быстрой установки и запуска под Windows (`install.bat`, `run.bat`).

Этот репозиторий подойдёт тем, кто:
- хочет увидеть «живой» пример структурированного Tkinter‑проекта;
- изучает ООП и разделение логики/GUI на практике;
- ищет основу для своего обучающего или развлекательного приложения‑викторины.

## Установка

1. Убедитесь, что установлен **Python 3**.
2. Клонируйте репозиторий:

git clone https://github.com/igormikt/millionaire-game-v3.git
cd millionaire-game-v3


3. (Рекомендуется) создайте и активируйте виртуальное окружение.
4. Установите зависимости:

pip install -r requirements.txt

## Запуск

Есть два основных способа запустить игру:

- Через командную строку:

python main.py


- В Windows двойным щелчком по файлу `run.bat` (после установки зависимостей).

После запуска откроется главное окно игры — можно сразу начинать отвечать на вопросы и «зарабатывать» виртуальные деньги.

## Структура проекта

- `main.py` — точка входа, запуск приложения и интерфейса.
- `modern_ui_interface.py` — реализация современного интерфейса на Tkinter.
- `game_logic.py` — игровая логика, уровни, проверка ответов.
- `questions.json` — база вопросов и вариантов ответов.
- `requirements.txt` — список сторонних библиотек.
- `install.bat`, `run.bat` — вспомогательные скрипты установки и запуска.
- `LICENSE` — файл лицензии (MIT).
